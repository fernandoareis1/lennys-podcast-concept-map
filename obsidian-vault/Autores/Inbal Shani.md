# Inbal Shani

Chief Product Officer do GitHub. Anteriormente General Manager na AWS e Microsoft, e engenheira sênior em Amazon Robotics. Lidera a visão de produto para GitHub Copilot, GitHub Advanced Security e a plataforma de desenvolvimento de software. Voz importante no debate sobre o futuro do desenvolvimento de software com IA, a relação humano-IA, e como estruturar times de inovação que chegam à produção.
