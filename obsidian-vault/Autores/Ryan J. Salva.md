# Ryan J. Salva
**Papel:** VP of Product at GitHub | Liderou a incubação e lançamento do GitHub Copilot | Ex-líder de produto sênior na Microsoft (One Engineering System, Azure DevOps) | Background em filosofia da estética e teoria crítica

Ryan J. Salva tem mais de 20 anos em produto e desenvolvimento de software, passando por startups e pela Microsoft antes do GitHub. No GitHub, coordenou a transição do GitHub Copilot de projeto experimental no GitHub Next (time de P&D) para produto operacional em escala global — um dos lançamentos de IA mais influentes na história do desenvolvimento de software. Copilot usa o modelo CodeX da OpenAI para oferecer autocomplete multilinha em tempo real no editor, com pesquisas apontando que ~40% do código Python de usuários ativos é gerado pelo produto. Formado em filosofia e literatura (filosofia da estética e teoria crítica do século XX), usa esse background para pensar sobre ética, linguagem e criatividade em produtos de IA.
