# Ryan J. Salva

VP de Product no GitHub. Responsável pela incubação e lançamento do GitHub Copilot — um dos primeiros produtos de AI para desenvolvedores em escala. Antes do GitHub, liderou produto na Microsoft (One Engineering System, Azure DevOps). Background incomum: filosofia da estética e teoria crítica do século XX, não engenharia.

## Episódios

## Conceitos principais
- [[Horizons de investimento]] — 5–10% para apostas ousadas/H3; ~30% para operações; ~60% para melhorias incrementais; split varia por maturidade e contexto
- [[Pesquisa Aplicada Não-Universitária — GitHub Next]] — Copilot nasceu do GitHub Next; knowledge transfer antes de mover pesquisadores de volta; product team deve sentir que controla o roadmap
- [[IA como Amplificador — Pair Programmer Invisível]] — AI remove drudgery para que desenvolvedor fique no flow criativo; par programador invisível; augmenta, não substitui
