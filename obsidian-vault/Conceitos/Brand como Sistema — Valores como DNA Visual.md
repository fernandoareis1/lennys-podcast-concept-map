# Brand como Sistema — Valores como DNA Visual
**Fluxo:** [[02 - Estratégia & Visão]] · **Tema:** [[Estratégia — Visão, missão & narrativa]] · **Camada:** L3
**Tipo:** Framework · **Fontes:** [[Barbra Gago]]

## Ideia central
Brand não é logo + cor + fonte. É um **sistema** onde cada elemento (forma, movimento, linguagem, fotografia) é uma extensão dos valores da empresa. Quando os valores estão embutidos no DNA visual, os colaboradores não precisam ser treinados para "entregar a brand" — ela emerge naturalmente em cada interação com clientes, parceiros e candidatos.

No caso do Miro, cada letra da palavra MIRO representava um valor diferente, com comportamento visual próprio: a letra que representava **agilidade** se movia de forma ondulante. O resultado é uma linguagem visual coesa que qualquer pessoa da empresa consegue usar consistentemente — não porque memoriza regras, mas porque o sistema é intuitivo e autorreferenciado.

## Como aplicar
1. **Mapeie os valores → atributos visuais**: para cada valor da empresa, identifique uma propriedade visual (cor, movimento, forma, textura). Pergunte: "se esse valor fosse uma coisa, como seria?"
2. **Construa um brand system, não um brand manual**: em vez de um PDF de regras, crie um sistema com componentes reutilizáveis e comportamentos definidos — como uma biblioteca de design, não uma lista de proibições.
3. **Integre valores no produto**: valores afetam como o produto age (ex.: "agility" → o Miro é extremamente flexível no canvas); o design visual deve refletir o que o produto já faz.
4. **Envolva toda a empresa no processo**: rebrand não é trabalho de marketing — é projeto com sprint teams, donos por área (sales deck, produto, eventos), e feedback de clientes early adopters ao longo do caminho.
5. **Nod à identidade anterior**: em um rebrand, mantenha algum elemento da identidade original para preservar o elo com quem já ama a marca (ex.: o amarelo do Post-it do RealtimeBoard virou o M amarelo do Miro).

## Insights por autor
### [[Barbra Gago]]
- "Os valores podem moldar como grande e quão fundo você vai com sua marca. Quando valores são fundação de como as pessoas se comportam na organização, você não precisa treinar cada touchpoint — ele simplesmente emerge."
- Miro: "Tínhamos cada letra da palavra Miro em uma forma diferente com características diferentes, ligadas a valores distintos. A letra que representava agilidade tinha formato ondulante e se movia de forma ágil."
- "Nosso objetivo era criar um sistema visual ou uma linguagem visual que pessoas de diferentes partes do mundo pudessem entender umas às outras sem necessariamente falar o mesmo idioma."
- Sobre rebrand como processo de produto: "Fizemos do rebranding uma abordagem de produto — sprint teams, agile, donos para cada parte. Era preciso que todos estivessem envolvidos de alguma forma."
- "Um brand forte e escalável se pensa mais como sistema: como ele se torna uma extensão de tudo que você vai fazer visualmente, ao falar com clientes, ao apresentar o produto."
