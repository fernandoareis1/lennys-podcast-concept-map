# Contratar para Elevar o Time

**Fluxo:** [[09 - Liderança & Times]] · **Tema:** [[Liderança — Modelos de time]] · **Camada:** L3
**Tipo:** Mindset
**Fontes:** [[Elizabeth Stone]]

## Ideia central
Contratação não deve preencher uma caixa — deve **elevar o time inteiro**. A pergunta certa não é "essa pessoa consegue fazer o trabalho?" mas "essa pessoa vai fazer com que todos ao redor dela sejam melhores?" Isso muda o critério de skills baseline (que a pessoa tem o mínimo necessário) para skills aditivas (o que ela traz que *ninguém no time atual traz*).

## Como operacionalizar
- **Pergunta favorita de Elizabeth para candidatos**: "O que você faria diferente se tivesse o meu trabalho?" — quem responde bem demonstra pensamento sistêmico, não ego; quem não responde bem provavelmente não vai elevar ninguém
- **Mapeie lacunas antes de contratar**: qual perspectiva está faltando no time? Qual skill estamos contornando em vez de ter?
- **Não contrate pela mediana**: se a pessoa é "ok para o papel", ela provavelmente mantém a densidade estável em vez de elevá-la
- **Olhe para quem essa pessoa vai tornar melhor**: não só o que ela vai produzir, mas quem ela vai puxar para cima

## A conexão com densidade de talento
Cada contratação é uma oportunidade de elevar o baseline do time. Cada contratação mediana dilui a densidade. Com o tempo, times que contratam "bom o suficiente" convergem para mediocridade — não porque as pessoas são ruins, mas porque o padrão coletivo caiu.

## Insights por autor
### [[Elizabeth Stone]]
- "I don't want someone who fits a box. I want someone who raises the bar for the whole team."
- Pergunta favorita de entrevista: "What would you do differently if you had my job?" — detecta quem pensa sistemicamente vs. quem está apenas tentando impressionar
- Contratação aditiva é especialmente crítica em liderança: um líder que eleva o time multiplica o efeito; um líder que não eleva limita toda a organização abaixo dele
- Elizabeth aplica o mesmo princípio para si mesma: toda vez que contrata alguém, pergunta se essa pessoa a tornará melhor pelo simples fato de estarem no mesmo time
