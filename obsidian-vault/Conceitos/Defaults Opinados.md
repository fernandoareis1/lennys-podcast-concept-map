# Defaults Opinados
**Fluxo:** [[07 - Ativação & Retenção]] · **Tema:** [[Retenção — Ativação & onboarding]] · **Camada:** L3
**Tipo:** técnica · **Fontes:** [[Adam Fishman]]

## Ideia central
Tornar difícil fazer a escolha errada e fácil fazer a escolha certa — sem eliminar a escolha. Baseado no aprendizado escalado de centenas de milhares de usuários, o produto pré-configura os melhores padrões (pricing, estrutura, configurações) como ponto de partida e aumenta o atrito para quem quiser sair do padrão. A autonomia do usuário é preservada; a trajetória de sucesso é projetada.

## Como aplicar
1. Identifique padrões de sucesso com dados (ex.: criadores com 3 tiers de preço têm maior LTV).
2. Defina defaults que replicam esses padrões para novos usuários.
3. Adicione fricção contextual (explicação, aviso) quando o usuário tenta desviar do default — não bloqueie, informe.
4. Produtize o aprendizado obtido via intervenção humana: o que o sucesso consegue manualmente vira default no produto.

**Patreon example:** preço mínimo sugerido de $3–5 (não $1), 3–5 tiers (não 40), estrutura de página opinionada — tudo pré-configurado com atrito extra para mudar.
