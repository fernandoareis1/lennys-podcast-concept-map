# Forças do Progresso (Push / Pull / Ansiedade / Hábito)

**Fluxo:** [[01 - Discovery & Pesquisa]] · **Tema:** [[Discovery — Jobs to be Done (teoria da demanda)]] · **Camada:** L3
**Tipo:** Modelo causal de comportamento
**Fontes:** [[Bob Moesta]]

## Ideia central
Toda troca de comportamento é resultado de **4 forças**. A mudança só acontece quando as forças a favor superam as contra:

- **F1 — Push (contexto):** a dor/situação que empurra para fora do produto atual.
- **F2 — Pull (atração):** a visão do novo resultado desejado.
- **F3 — Ansiedade do novo:** medo do desconhecido (mais features = mais ansiedade!).
- **F4 — Hábito do presente:** a inércia do que já se usa.

> Se **F1 + F2 não forem maiores que F3 + F4**, a pessoa não se move.

## Como aplicar
- Para vender mais, muitas vezes basta **reduzir fricção** (F3/F4) — não adicionar features.
- Ex.: incluir mudança + 2 anos de storage no preço do imóvel removeu a fricção e **+30% de vendas**.
