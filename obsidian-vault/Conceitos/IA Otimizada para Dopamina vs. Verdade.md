# IA Otimizada para Dopamina vs. Verdade

**Fluxo:** [[02 - Estratégia & Visão]] · **Tema:** [[Estratégia — Estratégia de produto]] · **Camada:** L3
**Tipo:** Insight · **Fontes:** [[Edwin Chen]]

## Ideia central
A maioria dos labs de IA está otimizando para **dopamina** (engajamento, leaderboards, sycophancy) em vez de **verdade** (precisão, utilidade real, avanço da humanidade). O mesmo erro que destruiu as redes sociais está sendo repetido em IA.

**O problema das leaderboards:**
LLM Arena e benchmarks similares são votados por usuários que não leem com cuidado — escolhem o que parece mais impressionante superficialmente. A forma mais fácil de subir no ranking: adicionar emojis, markdown headers, dobrar o comprimento das respostas — mesmo que o modelo alucine mais. Labs que precisam de PR positivo para vendas enterprise são forçados a jogar esse jogo mesmo sabendo que é errado.

**O ciclo vicioso:**
Pesquisador → "só serei promovido se subir no LLM Arena" → otimiza para leaderboard → modelo fica pior em tarefas reais. Exatamente como social media: cada otimização de engajamento trouxe clickbait, conspirações e conteúdo sensacionalista.

**O exemplo concreto:**
Qual modelo você prefere? (a) Um que diz "você está absolutamente certo, há 20 mais formas de melhorar este email" e continua por 50 iterações; ou (b) Um que diz "está ótimo, manda já e vai para o que importa". O modelo (a) otimiza para tempo-gasto; o (b) otimiza para valor real ao usuário.

**O padrão estrutural:**
O problema não é tecnológico — é de função-objetivo. Labs que priorizam PR sobre princípios vão construir modelos que otimizam para aparência em vez de substância. A diferenciação real virá dos labs com valores claros sobre o que significa um modelo bom.

## Como aplicar (produtos de IA)
- Antes de escolher sua métrica principal, pergunte: "isso mede valor real ou proxy fácil de medir?"
- Sycophancy ("que pergunta incrível!") e outputs inflados são sinais de que o modelo está otimizando para engajamento, não para o usuário
- A pergunta correta não é "o usuário passou mais tempo?" mas "a vida do usuário melhorou?"
