# Métricas de input vs. output
**Fluxo:** [[03 - Priorização & Roadmap]] · **Tema:** [[Priorização — Outcomes & metas]] · **Camada:** L3
**Tipo:** Framework · **Fontes:** [[Bill Carr]]

## Ideia central
**Output metrics** (receita, active customers, free cash flow) são os resultados que a empresa quer, mas ninguém consegue controlá-los diretamente. **Input metrics** são as variáveis controláveis que causam os outputs — seleção de produtos, preço, velocidade de entrega, qualidade da experiência.

O insight da Amazon: quando o fim de trimestre chega e os números ficam aquém, times correm para ações de curto prazo (promoções, emails extras, Price Days) que ou não funcionam ou puxam receita futura para o presente. A causa raiz é gestão pelo output. A solução é identificar o flywheel — os inputs que, se melhorados consistentemente, garantem que os outputs seguirão.

Prova: em ~2007, Jeff e o S-Team revisaram a lista de 500 S-Team goals e encontraram apenas 10 com métricas financeiras. Os outros 490 eram inputs mensuráveis da experiência do cliente. Foi uma escolha deliberada.

**Teste para identificar se é input metric**: (1) está sob o controle do time? (2) toca a experiência do cliente? (3) existe alguma hipótese de causalidade com um output desejado?

## Como aplicar
1. **Mapeie a experiência do cliente de ponta a ponta** — cada etapa do funil gera candidatos a input metrics (velocidade, qualidade, facilidade em cada momento).
2. **Identifique os inputs do seu flywheel**: quais são as 4-6 variáveis que, se melhoradas, nenhum cliente no futuro vai preferir piorar? (Preço menor, entrega mais rápida, mais opções — intemporais.)
3. **Evite compound metrics**: não some inputs numa única pontuação. Cada input deve ser gerenciado separadamente; combinações "obfuscam" as causas.
4. **Foque as metas em inputs durante o planejamento**: se os OKRs são todos outputs (receita, crescimento de users), você perderá a alavanca quando os números vacilarem.
5. **Mida cada input de múltiplas formas antes de escolher**: a Amazon mediu "seleção" errado por anos antes de refinar a métrica certa.
