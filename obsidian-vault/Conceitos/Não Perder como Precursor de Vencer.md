# Não Perder como Precursor de Vencer

**Fluxo:** [[10 - Founder & Carreira]] · **Tema:** [[Founder — Mentalidade de founder]] · **Camada:** L3
**Tipo:** Mindset · **Fontes:** [[Jason Droege]]

## Ideia central
A cultura tech celebra "vá em frente, quem se importa com o risco?" — mas os melhores empreendedores olham o perfil de risco de cada decisão e buscam apostas assimétricas positivas. Sobreviver é pré-condição para vencer: "você precisa sobreviver para que essa condição seja cumprida." Na jornada empreendedora, a mente diz que algo é "intransponível" — mas o mantra "o fim nunca é o fim" desbloqueia: sempre existe outra saída, mesmo que imperfeita. O erro não é tomar riscos — é tomar riscos sem entender o downside. Uma decisão de alto risco que dá errado pode cozer você completamente, e "muitas vezes não há saída." Ciclos de hype amplificam esse erro: todos em modo "vá em frente" ao mesmo tempo, quando os clientes só querem que você resolva os problemas deles e que ainda exista daqui a 5 anos para isso.

## Como aplicar
1. Antes de uma decisão grande, mapeie o worst case: se der errado, posso sobreviver? Existe saída viável?
2. Prefira decisões assimétricas: upside grande, downside limitado e reversível.
3. Em ciclos de hype, pergunte: estou tomando riscos porque é racional ou porque todo mundo está tomando?
4. Filtre ideias de novo negócio rápido: tem network effects? Receita recorrente? Margem que permite diferenciação? Se não, qual é a lógica de que vai valer bilhões?
5. Use o mantra "o fim nunca é o fim" quando a mente cria impasse — sempre existe a próxima ação, mesmo que não seja ótima.
