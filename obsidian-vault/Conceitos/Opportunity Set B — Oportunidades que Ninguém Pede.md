# Opportunity Set B — Oportunidades que Ninguém Pede

**Fluxo:** [[10 - Founder & Carreira]] · **Tema:** [[Founder — Carreira & progresso]] · **Camada:** L3
**Tipo:** Framework · **Fontes:** [[Jason Feifer]]

## Ideia central
Em qualquer posição, existem dois conjuntos de oportunidades. O **Opportunity Set A** é o que é pedido de você: suas responsabilidades formais, as expectativas do cargo, o que seu manager quer que você entregue. O **Opportunity Set B** é o que está *disponível* para você — mesmo que ninguém jamais vá pedir. É onde o crescimento real acontece. Focar exclusivamente no Set A te qualifica apenas para continuar fazendo o que já faz; o Set B é onde surgem novas identidades, novas direções e novas trajetórias.

## Os dois conjuntos
| Conjunto | Definição | Exemplos |
|---|---|---|
| **Set A** | O que é pedido de você | Responsabilidades do cargo, KPIs, expectativas do manager |
| **Set B** | O que está disponível, mas ninguém pede | Novos projetos, hobbies, speaking, escrever um livro, um podcast |

## Por que o Set B é infinitamente mais importante
- Se você só faz o Set A, você é **qualificado apenas para o que já faz** — não para o próximo nível
- Set B é onde você **amplia a identidade** para além do título atual
- O payoff raramente é imediato — pode vir meses ou anos depois, em forma de oportunidade que só apareceu porque você construiu algo antes
- Set B informa o Set A: o que você aprende fora do cargo inevitavelmente torna você melhor no cargo

## Como Jason Feifer aplicou
Quando se tornou editor-chefe da Entrepreneur, **ninguém pediu** que ele fosse ao circuito de palestras, que se tornasse bom em entrevistas de podcast ou que escrevesse um livro. Sua responsabilidade era fazer uma boa revista. Mas ao reconhecer o que estava *disponível*, ele criou uma carreira paralela como educator, speaker e author — e hoje se identifica não como "editor de revista" mas como "empreendedor que ajuda empreendedores". Essa identidade expandida só foi possível porque ele perseguiu ativamente o Set B.

## Como identificar seu Set B
1. Liste o que está ao seu alcance que **ninguém está pedindo** — projetos internos, comunidades, skills adjacentes, formatos de comunicação
2. Pergunte: qual dessas coisas, se eu fizer bem, pode abrir uma direção que o Set A nunca abrirá?
3. Não espere saber o payoff com antecedência — a maioria das oportunidades de Set B gera valor de formas imprevisíveis
4. Comece pequeno: a persistência ao longo do tempo supera o grande bet inicial

## A objeção mais comum: tempo
A falta de tempo é real. A resposta do framework é que Set B não exige grandes blocos de horas — exige **reconhecimento de oportunidade e ação consistente**. O podcast começou pequeno. O livro foi escrito aos poucos. A palestra apareceu uma vez e abriu mais portas.
