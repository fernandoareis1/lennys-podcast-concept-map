# Post-Training como Novo Frontier — Do Pré-treinamento ao Especialista

**Fluxo:** [[02 - Estratégia & Visão]] · **Tema:** [[Estratégia — Estratégia de produto]] · **Camada:** L3
**Tipo:** Insight · **Fontes:** [[Garrett Lord]]

## Ideia central
O pré-treinamento atingiu um plateau de retornos. Os ganhos marginais dos modelos frontier agora vêm quase inteiramente do **post-training** — RLHF, instruction-tuning, e dados de alta qualidade produzidos por especialistas humanos. Isso inverte radicalmente o perfil de quem é valioso: generalistas (mão de obra barata, tarefas simples) → **especialistas de domínio** (PhDs, mestres em STEM, direito, medicina, finanças).

## O que mudou
- Pre-training (2017–2023): escalar parâmetros + mais texto da internet → maiores ganhos
- Post-training (2024+): modelos já são bons demais para tarefas genéricas; precisam aprender nuance, raciocínio, tool-use especializado
- Tipos de dados que vêm a seguir: arquivos CAD, tool-use científico, descoberta de drogas, dados multimodais (áudio, vídeo), systems operacionais esotéricos
- "The type of data we're going to need is going to evolve a lot."

## Implicação estratégica
Qualquer negócio com **acesso a uma audiência de especialistas** tem um ativo estratégico desproporcional. Não é o dado em si — é o canal para produzir dados de alta qualidade sem custo de aquisição. Isso explica a corrida das frontier labs para fechar acordos com redes de profissionais (Handshake, Scale, Surge AI, etc.).
