# Product-Led Acquisition — Quatro Mecanismos
**Fluxo:** [[06 - Growth & Aquisição]] · **Tema:** [[Growth — Modelos de crescimento]] · **Camada:** L3
**Tipo:** Framework · **Fontes:** [[Julian Shapiro]]

## Ideia central
Product-Led Acquisition (PLA) é mais específico que PLG: é o mecanismo pelo qual o **uso natural do produto** adquire novos usuários — sem custo marginal, sem depender de terceiros (Google, Facebook). Enquanto programas de referral são incentivos artificiais ("faça X para ganhar Y"), PLA é orgânico: o usuário convida outros porque precisa fazê-lo para extrair valor do produto.

Julian Shapiro identificou 4 categorias de PLA, cada uma com compounding natural e resistência à volatilidade de canais pagos.

## Os 4 mecanismos

### 1. Quitar uma Dívida (Debt Settlement)
**Mecânica:** para o usuário A receber algo de usuário B, B precisa criar conta
**Exemplos:** Venmo (dividir conta), PayPal (pagamento de fornecedor), OpenSea (NFT)
**Por que é poderoso:** quem vai criar conta para não receber dinheiro que é seu? Conversão quase garantida
**Aplicação:** qualquer produto onde A envia algo de valor a B e B precisa de conta para reivindicar

### 2. Convidar para Conversa Exclusiva (Conversation Gate)
**Mecânica:** usuários convidam outras pessoas para conversas que só acontecem dentro da plataforma
**Exemplos:** Slack (coworkers + Slack Connect para vendors), WhatsApp/Telegram (amigos), Discord (comunidades)
**Insight sobre Slack Connect:** brilhante porque expande o PLA além dos colegas de trabalho, atingindo pessoas fora da organização
**Aplicação:** adicionar chat ou colaboração dentro de qualquer produto onde a conversa não pode acontecer fora dele

### 3. Billboarding
**Mecânica:** o produto se anuncia para si mesmo através do uso normal
**Subcategorias:**
- **Assinatura passiva:** Hotmail "Enviado via Hotmail", "Sent from my iPhone" — cada email enviado é um anúncio
- **Link compartilhado:** Calendly, Dropbox, GoFundMe — o usuário *tem que* compartilhar o link para usar o produto; cada compartilhamento é um anúncio
- **Objeto visível:** Tesla, AirPods, Nike — o produto é um outdoor ambulante pelo mundo físico
- **Sinal de status:** Twitter Blue ✓, Telegram Premium ★ — visível para todos na plataforma, gera curiosidade
**Custo:** zero. Escala: infinita. Quem faz: os próprios usuários.

### 4. UGC (User Generated Content)
**Mecânica:** usuários criam conteúdo que atraem outros usuários para dentro da plataforma
**Exemplos:**
- TikTok, Instagram, YouTube — conteúdo criado por usuários é compartilhado off-platform, trazendo espectadores para a plataforma
- Quora, Reddit, Stack Overflow — conversas indexadas pelo Google geram tráfego orgânico
- eBay, Etsy, Airbnb — listings criados por vendedores atraem compradores
**Pergunta-chave para qualquer produto:** "Usuários criam conteúdo dentro do produto? Como incentivamos o compartilhamento? A página de compartilhamento é otimizada para consumo?"

## Por que PLA supera referral programs
Programas de referral são incentivos artificiais para comportamento que de outra forma não ocorreria. PLA é o comportamento natural do produto. Diferença prática:
- Referral atrai "caçadores de recompensa" que não são usuários reais
- PLA atrai usuários que precisam do produto para resolver um problema real
- Referral tem CAC escondido; PLA tem CAC zero
- Referral não tem compounding; PLA tem (quanto mais usuários, mais orgânico)
