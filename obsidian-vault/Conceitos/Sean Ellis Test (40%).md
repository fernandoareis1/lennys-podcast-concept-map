# Sean Ellis Test (40%)

**Fluxo:** [[01 - Discovery & Pesquisa]] · **Tema:** [[Discovery — Descoberta contínua & validação]] · **Camada:** L3
**Tipo:** Métrica de validação de PMF
**Fontes:** [[Sean Ellis]] · [[Jag Duggal]]

## Ideia central
Uma pergunta: **"Como você se sentiria se não pudesse mais usar este produto?"** (muito decepcionado / um pouco / não decepcionado). Se **≥40%** dizem "muito decepcionado", é um forte **indicador antecipado** de product-market fit. Retenção é o indicador definitivo, mas exige tempo e dados — o teste dá sinal já no dia 1.

## Como aplicar
- Foque nos **must-have users** (os "muito decepcionados"); **ignore** os "um pouco decepcionados" — ouvi-los dilui o produto para quem realmente ama.
- Funciona já num MVP. Caso Lookout: de 7% → 40% em 2 semanas, ao focar no que o núcleo (antivírus) valorizava.

### [[Jag Duggal]]
- O Nubank usa o test como **portão obrigatório antes de escalar qualquer produto** — nunca escalam até atingir o threshold.
- Calibração cultural: brasileiros são mais educados em respostas de pesquisa, então o Nubank elevou o threshold de 40% para **50%** para compensar o bias de polidez.
- Usar com [[Segmentação Bullseye — Produto que Falhou]]: quando o score geral não atinge o threshold, busque o subsegmento que ama o produto — esse é o wedge.
