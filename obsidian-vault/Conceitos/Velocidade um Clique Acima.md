# Velocidade um Clique Acima
**Fluxo:** [[09 - Liderança & Times]] · **Tema:** [[Liderança — Liderança & founder]] · **Camada:** L3
**Tipo:** Framework · **Fontes:** [[Claire Vo]]

## Ideia central
Toda organização tem uma velocidade natural de operação. Essa velocidade tende a se acomodar no pior anti-padrão possível: o ritmo das reuniões recorrentes. O antídoto é simples e tático: sempre que você for definir um prazo, puxe-o um nível para cima.

**A regra do relógio:**
- Se você acha que precisa ser feito este ano → este semestre
- Este semestre → este trimestre
- Este trimestre → este mês
- Este mês → esta semana
- Esta semana → hoje
- Hoje → nesta reunião

Não é sobre trabalhar mais. É sobre questionar o prazo padrão antes de comprometer-se com ele.

**O anti-padrão: reuniões como árbitro de timelines:**
"Vamos discutir isso na próxima reunião" é uma timeline artificial imposta pelo Google Calendar, não pela realidade. Quanta informação eu realmente preciso para decidir isso? Quando posso ter isso? Qual é o verdadeiro prazo necessário — não o calendário conveniente?

**SLA pessoal como alavancagem:**
Claire nunca quer ser gargalo. Em posições de liderança, você é ponto natural de decisão, desempate, aprovação. Se sua SLA pessoal é lenta, toda a sua organização fica represada. Resposta rápida é responsabilidade de liderança, não cortesia.

**Expectativa explícita, não implícita:**
Velocidade não é imposta por pressão — é comunicada como expectativa. "Minha expectativa é que você traga o clock speed um clique acima." Quando a expectativa é explícita, os líderes calibram para ela. Quando é implícita, cada um calibra para o próprio conforto.

## Como aplicar
1. **Audite os próximos 5 deadlines do seu time:** cada um passa no teste do "one click faster"?
2. **Elimine o "vamos ver na próxima reunião":** substitua por "quando você pode ter isso? Posso revisar em 24h?"
3. **Defina seu SLA pessoal:** e-mails, pull requests, decisões — qual é o seu tempo de resposta padrão? Torne-o explícito
4. **Separe informação de decisão:** "quanto eu preciso saber para decidir isso?" é diferente de "quando é a próxima reunião?" Use o primeiro para definir o prazo
5. **Cascateie:** comunique a expectativa explicitamente para sua liderança direta. Velocidade como norma se propaga de cima para baixo
