# Visão como Fotografia do Futuro

**Fluxo:** [[02 - Estratégia & Visão]] · **Tema:** [[Estratégia — Visão, missão & narrativa]] · **Camada:** L3
**Tipo:** Framework · **Fontes:** [[Ebi Atawodi]]

## Ideia central
Visão é uma **fotografia do futuro** — não é missão, não é estratégia. Você avança 5 anos no tempo, tira uma foto do que existe, volta para o presente e usa essa imagem para decidir o que construir hoje.

**Quatro elementos de uma visão poderosa:**
1. **Lofty** — aspiracional o suficiente para inspirar e motivar o time
2. **Realistic** — alcançável dentro de um horizonte plausível (não ficção científica)
3. **Devoid of current constraints** — livre das limitações técnicas/organizacionais atuais; pensa no que o mundo poderia ser, não no que pode fazer hoje
4. **Grounded in a potent user problem** — ancorada em um problema real e significativo de usuário

**Visão ≠ Missão:**
- Missão = o porquê eterno (ex: "democratizar transporte")
- Visão = a fotografia concreta do futuro em 5 anos

**Exemplo clássico:** "Uber's vision was a city without parking." Não começa com feature; começa com como o mundo será diferente para o usuário.

## Como aplicar
1. Identifique o problema potente de usuário que o produto resolve
2. Projete-se 5 anos no futuro: se tivéssemos sucesso total, como seria o dia-a-dia do usuário?
3. Descreva essa cena sem mencionar features ou limitações atuais
4. Verifique os 4 critérios: lofty, realistic, sem constraints atuais, problema de usuário
5. Use essa fotografia como filtro de priorização: "isso nos aproxima da foto?"

## Insights por autor
### [[Ebi Atawodi]]
- "A vision is like taking a photograph of the future. You fast-forward five years out, come back to tell us what to fix and put in place."
- Os quatro elementos devem todos estar presentes: se é apenas aspiracional mas não grounded in a user problem, fica vago; se é grounded mas não lofty, não inspira.
- Visão é diferente de missão — missão é o "porquê" perene; visão é o estado futuro concreto.
- Exemplo Uber: "a city without parking" — não é feature, é fotografia do mundo transformado.
